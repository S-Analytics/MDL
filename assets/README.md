# MDL Application Assets

This directory contains application assets for the MDL project.

## Icon Files

- `icon.svg` - Source vector icon (512x512)
- `icon.png` - Application icon (512x512)

## Usage

The icons are used for:
- Web application favicon
- Documentation and branding
- General application identity

## Note
The current icons are placeholders. For production, create custom icons at 1024x1024 resolution for optimal quality across different display sizes.
